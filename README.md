# Repositori Tugas-tugas PBI
## Untuk tugas setiap minggu dipisah dalam branch masing-masing

Tugas 1
```
Membangun Struktur Dasar Website dengan HTML

Detail Tugas
Dalam halaman utama website wajib terdapat beberapa struktur, diantaranya:
1. Header dengan navigasi
2. Hero section dengan deskripsi utama
3. Section layanan/produk/konten utama
4. Section informasi atau testimoni (jika ada)
5. Formulir kontak sederhana
6. Footer dengan informasi tambahan
Selain struktur di atas, peserta bebas menambahkan struktur lain sesuai dengan kebutuhan.
Poin yang dinilai:
Struktur HTML rapi dan menggunakan elemen yang tepat (40%)
Kelengkapan elemen dalam halaman web (30%)
Konsistensi dan kerapihan kode (30%)
Tools
Text Editor: Visual Studio Code / Sublime Text / Notepad, dll.
```

Tugas 2
```
Mengimplementasikan CSS pada Halaman Web

Detail Tugas
1. Tambahkan style CSS tugas yang telah dikerjakan
2. Implementasikan beberapa tools CSS yang telah dipelajari. Peserta dapat menambahkan hover pada tombol atau hyperlink
Poin yang dinilai:
1. Penggunaan tools CSS (40%)
2. Variasi tools CSS yang digunakan (25%)
3. Konsistensi desain antar halaman (35%)
Tools
Text Editor (Visual Studio Code, Sublime Text, Notepad, dll)
```

Tugas 3
```
Mengimplementasi Javascript pada Halaman Web

Detail Tugas
Peserta wajib menambahkan minimal dua fitur interaktif, seperti:
1. Toggle menu navigasi untuk tampilan mobile
2. Validasi form input pada halaman kontak
3. Efek animasi saat pengguna menscroll halaman
Pastikan juga JavaScript berjalan tanpa error
Poin yang dinilai:
1. Penguunaan tools Javascript (40%)
2. Logic yang dibuat (35%)
3. Hasil slideshow yang dijalankan (15%)
4. Konsistensi fungsi pada berbagai halaman (10%)
Tools
Text Editor (Visual Studio Code, Sublime Text, Notepad, dll)
```

Tugas 4
```
Deployment Website

Detail Tugas
Sebelum melakukan deploy, kamu harus pastikan bahwa:
1. Desain sudah rapi
2. Navigasi mudah dan dapat digunakan dengan baik
3. Interactive feature berjalan dengan baik
4. Struktur file rapi.
Gunakan salah satu platform hosting untuk deployment, jika sudah kemudian lakukan pengecekan
Peserta akan diminta untuk mempresentasikan hasil website yang sudah dikembangkan selama 4 minggu
Poin yang dinilai:
1. Website berhasil di-deploy dan dapat diakses melalui URL publik (40%)
2. Desain dan tata letak tetap rapi setelah di-deploy (30%)
3. Fitur interaktif dan navigasi berfungsi dengan baik (20%)
4. Struktur file di repository GitHub terorganisir dengan baik (10%)
Tools
Text Editor (Visual Studio Code, Sublime Text, Notepad, dll), Version Control, dan Hosting
```
