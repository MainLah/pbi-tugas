# Repositori Tugas-tugas PBI
## Untuk setiap minggu dipisah dalam branch masing-masing

Tugas 1
```
Detail Tugas
Dalam halaman utama website wajib terdapat beberapa struktur, diantaranya:
1. Header dengan navigasi
2. Hero section dengan deskripsi utama
3. Section layanan/produk/konten utama
4. Section informasi atau testimoni (jika ada)
5. Formulir kontak sederhana
6. Footer dengan informasi tambahan
Selain struktur di atas, peserta bebas menambahkan struktur lain sesuai dengan kebutuhan.
Poin yang dinilai:
Struktur HTML rapi dan menggunakan elemen yang tepat (40%)
Kelengkapan elemen dalam halaman web (30%)
Konsistensi dan kerapihan kode (30%)
Tools
Text Editor: Visual Studio Code / Sublime Text / Notepad, dll.
```

Tugas 2
```
Detail Tugas
1. Tambahkan style CSS tugas yang telah dikerjakan
2. Implementasikan beberapa tools CSS yang telah dipelajari. Peserta dapat menambahkan hover pada tombol atau hyperlink
Poin yang dinilai:
1. Penggunaan tools CSS (40%)
2. Variasi tools CSS yang digunakan (25%)
3. Konsistensi desain antar halaman (35%)
Tools
Text Editor (Visual Studio Code, Sublime Text, Notepad, dll)
```
